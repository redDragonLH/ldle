# 差分数组

可以视为前缀和的逆运算

```
    前缀和：对于一个给定的数列 A，它的前缀和数列 S 中 s[i] 表示从第 1 个元素第 i 个元素的总和。用公式表示为：
```

## 定义
对于给定的数列A,它的差分数列 B中 B[i]表示：原数列A中第i个元素与第i-1个元素的差
```
公式：
    B[1]= A[1]
    B[i] = A[i] - A[i-1],(1<=i<=n)
```
前缀和与差分是一对互逆运算
    如果数列 A 的差分序列是 B，那数列 B 的前缀和就是 A。
